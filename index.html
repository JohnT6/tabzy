Tabzy.js

Một thư viện tab đơn giản và siêu nhẹ được xây dựng bằng JavaScript thuần.

Dùng Thử

Tabzy là một thư viện tab siêu nhẹ được xây dựng bằng JavaScript thuần. (Mã nguồn có sẵn trên GitHub)

  • Không phụ thuộc (No dependencies)
  • Tùy chỉnh hoàn toàn bằng CSS
  • Hỗ trợ lưu trạng thái trên URL
  • API đơn giản và trực quan
  • Siêu nhẹ và được tối ưu hóa

Cài Đặt

Phương pháp Các bước
Tải Trực Tiếp Tải file ZIP

Chỉ cần giải nén file ZIP và sử dụng các file tabzy.min.js và tabzy.min.css trong dự án của bạn.

Tabzy cung cấp một file CSS nhỏ mà bạn có thể nhúng trực tiếp thông qua link CDN. Điều này giúp dễ dàng tích hợp vào dự án của bạn và tùy chỉnh khi cần thiết.

Nhúng file JavaScript của Tabzy vào dự án của bạn:

<script src="https://cdn.jsdelivr.net/gh/sondnpt00343/tabzy@v1.0.0/tabzy.min.js"></script>
Bạn có thể sử dụng các phiên bản Tabzy đã được phát hành tại đây.

Cách Sử Dụng

Đây là cấu trúc HTML cơ bản bạn cần để thiết lập Tabzy:

<ul id="my-tabs">
    <li><a href="#panel1">Tab 1</a></li>
    <li><a href="#panel2">Tab 2</a></li>
</ul>
<div id="panel1">Nội dung 1</div>
<div id="panel2">Nội dung 2</div>

Và đây là cách bạn có thể sử dụng nó với JavaScript:

// Khởi tạo một instance tab mới
const tabs = new Tabzy('#my-tabs', {
    activeClassName: 'tabzy--active',
    remember: true, // Giữ trạng thái tab đang hoạt động trên URL
    onChange: function({ tab, panel }) {
        console.log(`Đã chuyển sang ${tab.textContent}`);
    }
});

// Chuyển sang một tab cụ thể
tabs.switch('#panel2');

// Hủy bỏ instance tab
tabs.destroy();

Tùy Chọn

Tùy chọn Kiểu Mô tả
activeClassName string Class CSS được áp dụng cho tab đang hoạt động (mặc định: 'tabzy--active')
remember boolean Lưu trạng thái tab đang hoạt động trên URL (mặc định: false)
onChange function Một hàm callback chạy mỗi khi tab đang hoạt động thay đổi. Nó nhận vào một object với hai thuộc tính: tab (phần tử DOM của tab mới được kích hoạt) và panel (phần tử DOM của panel nội dung tương ứng). Theo mặc định, nó được đặt là null, nghĩa là không có hành động nào được thực hiện trừ khi bạn định nghĩa một hàm.

API

Phương thức Mô tả
switch(input) Chuyển sang một tab cụ thể dựa trên input được cung cấp. input có thể là: (1) một chuỗi đại diện cho thuộc tính href của tab (ví dụ: '#panel2'), hoặc (2) một phần tử DOM là một trong các link tab (ví dụ: một thẻ <a> từ danh sách tab).
destroy() Loại bỏ tất cả chức năng của Tabzy, khôi phục lại HTML gốc và gỡ bỏ các sự kiện.
